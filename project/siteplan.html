PC Build Help & Troubleshooter

Site Name

PC Build Help & Troubleshooter is a name that clearly describes what this site will help do, which is to teach beginners and help them learn about troubleshooting PCs

Site Purpose

PC Build Help & Troubleshooter is a beginner friendly website to help teach users about computer parts, computer building, basic upgrades, common repair issues, and troubleshooting. The website also provides interactive tools such as PC part and PC recommendation tools, a troubleshooting checklist, and a contact/help request form.

Scenarios

  • What PC parts do I need for my first build?
  • What steps should I take for troubleshooting a frozen PC?

Color Schema

The website will use the following colors

    Text -- will be used for all text
